BOILERPLATE FEATURES

List of prebuild features

Features are Plug & Play, means each feature can be added or removed easily to keep the application light weight.

Authentication

Sign up, login, password reset, OTP verification, and social logins (Google, Facebook, Twitter) - all authentication features ready to use out of the box

Live Chat

Real-time chat functionality with message history, typing indicators, file sharing, emojis, and group chat capabilities built right in

User Management

Complete user profile management, roles and permissions, and user-specific settings - everything you need to handle users effectively

Theme & UI Components

Modern, responsive UI components with customizable themes, dark mode support, and accessibility features - all designed to match your brand

New module creation in seconds

Generate complete CRUD functionality with just one command - includes models, controllers, routes, and UI components, saving hours of development time

Dynamic Input fields

Ask onboarding questions with a simple JSON configuration. JSON contains the questions and information like what input controls to show and input fields will render fully customizable with your theme

Payment Options

Complete payment system with Stripe integration, subscription management, usage-based billing, and invoice generation all ready to use

Settings

Control everything through settings, a separate setting for each module, device settings - light mode, dark mode, left drawer, font-size, user settings: update user info

Multi-language Support

Built-in internationalization support with easy language switching, RTL support, and customizable translations for all interface elements

Activity Log

Audit logs of the actions being performed. Nicely display activity logs to users.

Rebranding

Easily change the app name, package name, and project name by using a simple command. All references are updated in minutes including file names and import statements.

In-app Review

Easily configure the rules of when the in-app review popup should appear. Get good feedback from happy customers at the right time.

Multi-platform Support

Works on all desktop and Linux platforms. Improved performance for web. New modules are by default compatible with web and bigger screens. Deep linking of hrefs, SEO, and caching images done to improve web performance.

Auto upgrade app to latest version

Put your app on auto-pilot. Automatically upgrade the app to the latest version without going through app stores. Ensure all users are on the latest version with bug fixes and new features.

Google analytics integration

Track user behavior and app performance with built-in custom events for Google Analytics. Gain insights into user engagement, retention, and conversion to optimize your app experience.
