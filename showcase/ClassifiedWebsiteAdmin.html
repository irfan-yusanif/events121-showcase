Manage Brand & Models

You can allow customers to post ad by just asking title and descrption, but if you want to go deeper in each category, you can define brands/models or subcategories in each category. Like in mobile you can add Apple brand and can add all apple models

Edit a brand/model for SEO

Easily upload logo for your brand model Configuration for compressing images, decreasing storage costs Write dynamic content e.g, the put {firendlyLocation} in content where you want to show location, and it will be populated based on customer's selected location filter.

Monitor Ads

Review latest posted ads and easily delete spam ads and block users Admin can see chats in case of any spam or incident required by law enforcement agencies Admin can change ad category if its not posted in its appropiate category The ownership of ads can also be transfered from one user to another, and it will also transfer chats and other data to new user account.

collect user data and Grow business

You can show a popup is shown on home screen (near to a season sale) where users interested in buying bakra can input their contact information. And they'll later receive a call from support team Data is automatically grouped by phone numbers so support team can easily manage

Manage Locations

Add countries, regions, cities, famous places in a city Manage longitude & latitude to show ads on maps Each location is assigned points used to identify how popular it is. The location that are searched more has more points.

Custom Fields for Ad Categories

If you plan to go deep in each classified category, you can easily add more fields to each category. e.g, if someone wants to post ad of car, you can ask addition info "kms Driven", "Fuel Type". You can also set pre-defined values and enforce the rules. e.,g Fule Type must be Petrol, Diesel, CNG, Hybrid or Electric With a single click, duplicate the custom field and assign to other categories Configure how a custom field should appear in search filters.

Manage Users

Easily search and Block/UnBlock users Delete all ads for a spam user Assign a role to user or Revoke permissions See the list of active admin See the list of blocked users User is informed by email if their ad is deleted or user is blocked and they can appeal for review

System or User preferences

System level or default user level configurations are sent in preferences module. A use case is a list of spam detector keywords, when a new ad is posted, if ad contains any of the spam detector keywords, its automatically marked as spam, made hidden from public and is sent to admin for review.

Blog

A fully functional blog, you can create new blog posts, set SEO keywords and meta description, assign tags to posts and lot more. Grow your SEO with content enricher build in blog

Marketing log

Easily schedule marketing compaigns keep a track of all email sent to a customer so you don't overload them with lot of emails

Premium Packages

Convert your passion into revenue with full fledged premium packages Easily create new packages and configure how much to charge See the ads that have premium status Premium ads appear on top in search and can also have the tags like "Verified", "Urget" See the payment details and transactions

SEO Tools

SiteMap generator for Google Search Console - once you update/add new categories or brand/models you can generate the sitemap.xml with a single click. Search keywords - whenever user search something in website, its saved in database,

Manage Background Jobs

Backend processes can easily be managed through admin portal. UseCase 1 - when website is newly launched there might not be any views for ads so you don't want users to see ads posted days or weeks ago. The background job will auto review the ads so they look fresh. UseCase 2 - when you site is new and there are no ads posted. Use a webscraper that will run every hour and will copy the latest ads posted on other classfied websites within that hour

Email templates

10+ email premium templates ready to use, easily setup from admin portal
